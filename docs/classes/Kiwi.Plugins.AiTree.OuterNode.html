Kiwi.Plugins.AiTree.OuterNode Class

Defined in: src/ai-tree-1.0.0.js:188
Module: Plugins
Parent Module: Kiwi

Template for all responsive nodes. An Outer Node performs some process (usually an action or a condition check) then returns a STATUS to its caller. This STATUS dictates the entire tree traversal.

All nodes are based on the Outer Node.
